Comparación entre distintos frameworks

Rendimiento

  • Angular: El tamaño de este framework es de aproximadamente 500 KB lo que lo hace inadecuado para aplicaciones livianas, sin embargo, este framework ofrece muchísimas cosas a los desarrolladores desde plantillas hasta utilidades de prueba.
  • React: El tamaño de React es de 100 KB lo que lo hace ideal para aplicaciones livianas, esto es debido a que no incluye tantas cosas como Angular.
  • Vue: El tamaño de Vue es de 80 KB lo que lo hace ideal para aplicaciones livianas lo que hace que sea el mejor framework para aplicaciones livianas.
